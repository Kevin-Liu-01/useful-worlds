## Hermes Machines

Project URL: https://github.com/Kevin-Liu-01/hermes-machines

Skills: Open Source, TypeScript, Dedalus SDK, Cursor SDK, MCP, Python, Hermes Agent, Next.js 15

A persistent Hermes Agent deployed to a Dedalus microVM, with a bundled skill library, scheduled crons, and the Cursor TypeScript SDK wired in as an MCP server so it can spawn real coding agents that inherit the rig's conventions as `.cursor/rules`.

- Collapses an OpenAI-compatible chat endpoint, an always-on agent with persistent memory, and a code-execution environment into one durable surface.
- Cursor SDK bridge: 4 MCP tools (`cursor_agent`, `cursor_resume`, `cursor_list_skills`, `cursor_models`) so Hermes can delegate code tasks.
- 13 bundled skills, 4 pre-seeded cron automations, full lifecycle CLI (`deploy`, `chat`, `wake`, `sleep`, `destroy`).
